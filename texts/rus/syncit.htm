Коллективная работа немыслима без средств, позволяющих управлять задачами, распределять их между участниками команды, отслеживать изменения. Для этого используются модули трэкинга задач таких популярных систем коллективной работы, как Trac, Team Foundation Server, Salesforce и т.д. В то же время, человеку удобнее хранить все свои задачи, как личные, так и служебные, в персональном органайзере на своем компьютере или КПК. Для этой цели хорошо подходит MS Outlook 2007.

Между этими подходами нет противоречия, задача в том, чтобы синхронизировать "коллективное" и "персональное" хранилища задач. Эта проблема может быть решена с помощью нашей разработки SyncIT. Это маленькая программа, постоянно запущенная в system tray, которая периодически загружает информацию о новых \ изменившихся задачах из "коллективного" хранилища в "персональное" - список задач Outlook 2007. При этом, сам Outlook может быть не запущен.

В текущей версии поддерживаются следующие источники задач:

  • Trac
  • Google Code
  • dotProject
  • 1C Предприятие
В дальнейшем планируется расширить набор поддерживаемых источников. Планируется так же реализация обратной синхронизации - публикация на сервере изменений, произведенных с помощью Outlook.

Внедрения

Заказчик: Бизнесинтервест, г. Хабаровск

Сфера деятельности: автоматизация индустрии отдыха и развлечений

Задача автоматизации: вся информация по клиентам Бизнесинтервест и задачи по ним находятся в корпоративном хранилище данных на основе 1С Предприятия 8. Обслуживание клиентов осуществляют мобильные агенты, использующие iPhone для доступа к необходимой информации. Задача состоит в том, чтобы синхронизировать задачи в календаре iPhone со списком задач в корпоративном хранилище.

Решение: поскольку решения, позволяющего напрямую синхронизировать данные между 1С и iPhone, пока не существует, был выбран вариант с использованием календарей MS Outlook 2007 и Google Calendar. Для обмена информацией между Outlook и Google Calendar, а так же между Google Calendar и iPhone Calendar, использованы стандартные средства. Синхронизация между списком событий в 1С Предприятие и календарем Outlook осуществляется с помощью AnjLab.SyncIT. Для этой цели была разработана дополнительная функциональность SyncIT, позволяющая выбирать приемник данных Outlook (список задач либо календарь) и шаблон для формирования текста задачи.
