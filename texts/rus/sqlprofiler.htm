Семейство SQL Server 2005 включает бесплатную Express Edition. Экспресс версия является полнофункциональной, но не включает многие клиентские средства, в частности, SQL профайлер, что делает ее не очень удобной для использования при разработке.

Мы разработали свой профайлер, который может быть использован как с Express, так и с другими редакциями SQL Server. По функциональности он аналогичен стандартному и использует стандартные объекты SQL Server.
