Off.Grid:Electric предлагает полную техническую, операционную и финансовую модель, которая делает возможным применение высококачественных возобновляемых источников электроэнергии, доступных по цене в бедных странах. Это модель, которая придет в миллионы домов. Мы не продаем гаджеты или светильники, мы продаем услугу поставки электроэнергии за небольшую абонентскую плату.

Off.Grid:Electric Management System

Система, которая помогает Off.Grid:Electric вести бизнес. Она состоит из двух компонент:

  • Приложение для Android для мобильных агентов, которые работают с клиентами/li>
  • Веб-ориентированная подсистема центрального офиса для управления складом, бухгалтерией, агентами и клиентами. Подсистема так же предоставляет средства анализа и отчетности.
Мобильное приложение синхронизирует данные с центральной базой, но может использоваться независимо.

Подсистема центрального офиса размещена на "облачных" сервисах (Heroku и Zoho), что существенно снижает стоимость владения.

Технические детали:

  • Мобильная платформа: Android
  • Мобильная технология: Java
  • Веб технология: Ruby
  • Отчетность: Zoho Reports
  • База данных: PostgreSQL
