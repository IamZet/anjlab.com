Hunter - распределенная система для массовой закупки книг на торговых площадках в Интернет. Hunter это "брат" и расширение Harvester - системы для сбора информации с торговых площадок. Используя данные по имеющимся предложениям, и, задаваемый пользователем, список книг для покупки (включая желательную цену), Hunter отбирает для пользователя лучшие предложения. Предложения могут быть дополнительно отфильтрованы с использованием специальных правил, определяемых пользователем, например, "исключить предложения определенного продавца". Пользователь имеет возможность купить одно или более из отобранных предложений, непосредственно из интерфейса Hunter, без визита на торговую площадку. Заказанные покупки помещаются в очередь и покупаются автоматически, используя API площадки. На сегодняшний день, Hunter поддерживает покупку на Amazon.com, AbeBooks, Half.

Hunter спроектирован для максимального использования механизмов виртуализации, система может использовать сети виртуальных машин, либо "облачные вычисления" (computing clouds), в частности, Amazon Elastic Computing Cloud.

Внедрения

Заказчик: BetterWorldBooks, штат Индиана, США.

Бизнес: сбор новых и подержанных книг и их перепродажа через интернет. Продажи осуществляются на 19 рынках, включая Amazon.com, Ebay, и собственный магазин betterworld.com

Внедрение: начиная с 2009, как часть программы покупки книг для дальнейшей перепродажи ( Buy Back Program), BetterWorldBooks использует Hunter для закупки лучших предложений.
