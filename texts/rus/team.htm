AnjLab - команда профессиональных разработчиков программного обеспечения, имеющая большой опыт разработки коммерческих проектов для компаний по всему миру.

Мы специализируемся в создании следующих видов программного обеспечения:

  • Корпоративные решения
  • Web системы
  • Мобильные приложения
  • Системы телеметрии
  • Анализ данных и интеграционные решения

Наши основные технологические платформы::

  • .NET
  • Java
  • iOS, Android
  • Ruby on Rails, Phyton
  • Oracle, MS SQL Server, MySQL, PostgreSQL

Мы поддерживаем все стадии разработки программного обеспечения, включая постановку задачи (анализ), тестирование и поддержку продукта. В различных ситуациях схема работы может быть адаптирована, например, возможны варианты работы по готовым спецификациям или работа в качестве субподрядчика.

Мы принимаем активное участие в open source коммьюнити. Наша страница на GitHub.com.

Более подробная информация в разделах Команда, Партнеры и Заказчики и Проекты.
