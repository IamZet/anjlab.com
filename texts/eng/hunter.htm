Hunter is web based system designed for mass purchasing books on internet markets. Hunter is 'brother' and extension of Harvester - distributed system for collecting data of internet marketplaces. Using gathered data and wish list of book ISBNs and prices, Hunter proposes to user the best offers for each ISBN. Offers can be filtered by special rules defined by user, for instance, 'exclude international editions'. User is able to purchase one or more offers via Hunter web interface directly, without visiting marketplace. Orders are placed into purchasing queue and purchased automatically, using market API. For now Hunter supports purchasing on Amazon.com, AbeBooks, Half.

Harvester is specially designed for using benefits of vitalization. The system can use resources of virtual machines or resources provided by 'computing clouds' services, for instance, Amazon Elastic Computing Cloud.

Success story

Customer: BetterWorldBooks, IN, USA.

Business: collecting and re-selling books online through19 online marketplaces, including Amazon.com, Ebay, and their own online e-commerce site, betterworld.com

Story: since 2009, as part of their Buy Back Program, BetterWorldBooks uses Hunter to purchase books on internet marketplaces for further re-selling.
