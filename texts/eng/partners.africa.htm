Off.Grid:Electric Management System

The system which helps Off.Grid:Electric do their business. It consists of two subsystems:

  • Android based application for agents who work with customers
  • Web based backend system to manage warehouse, accounting and customers. It also provides reporting and analytical tools
Mobile application synchronizes data with backend, but able to work offline.

Backend subsystem is hosted in clouds (Heroku and Zoho), which reduces ownership cost drastically.

Technical details:

  • Mobile platform: Android
  • Mobile technology: Java
  • Web technology: Ruby
  • Reporting: Zoho Reports
  • Database: PostgreSQL
