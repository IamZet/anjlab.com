Developing

We support and develop several open source projects which are hosted on Google Code:

  • SQL Profiler для SQL Server 2005 - fully functional profiler for SQL Server 2005. It is analogous to standard profiler and supports all SQL Server editions, including Express Edition.
  • SyncIT - tasks synchronizer between personal and collective task management systems. It uses MS Outlook 2007 as 'personal' storage, set of supported 'collective' systems includes Trac, Google Code (Tickets), dotProject, 1C Enterprise 8. More supported systems are coming soon.
The projects are free and can be used 'as is'. However if you want to donate authors, please click advertising links located on project pages at Google Code.
